# Datary - Synthetic Data Generator

![Datary Logo](frontend/assets/logo.svg)

**Your data. Your structure. Generated.**

Datary is a professional synthetic data generator that helps you create realistic datasets for testing, development, and analysis.

## Features

- 🔓 **No account required** - Start generating instantly
- 🎯 **Realistic mock data** - Data that feels real
- 📦 **Multiple formats** - CSV, JSON, SQL, XML, YAML, HTML, Markdown
- ⚡ **Fast & easy** - Full control, no friction
- 🎨 **Beautiful UI** - Modern, dark theme optimized for developers
- 🌍 **Multiple locales** - Support for different countries and languages

## Tech Stack

### Backend
- Node.js + Express
- Faker.js for data generation
- JSON2CSV, XLSX for exports
- Rate limiting & security middleware

### Frontend
- Vanilla HTML, CSS, JavaScript
- Modern dark theme design
- Responsive for all devices
- Syntax highlighting for code preview

## Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

```bash
cd backend
